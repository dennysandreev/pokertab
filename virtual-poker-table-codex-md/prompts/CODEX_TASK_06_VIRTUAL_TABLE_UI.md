# CODEX TASK 06 — Virtual Table UI

## Goal

Build Telegram Mini App UI for virtual table.

## Tasks

1. Add Create Virtual Table screen.
2. Add Virtual Waiting Room.
3. Add Join Virtual Table screen.
4. Add Virtual Table main screen.
5. Add player seats.
6. Add board/community cards.
7. Add current player's private cards.
8. Add legal action buttons.
9. Add Raise modal.
10. Add All-in confirmation.
11. Add Admin Panel.
12. Add Raise Blinds modal.
13. Add Sit-out modal with auto-check and auto-fold checkboxes.
14. Add Sitting Out state.
15. Add Hand History screen.
16. Add loading/error states.

## Acceptance criteria

- UI works on mobile Telegram Mini App viewport;
- current actor is obvious;
- only legal actions are clickable;
- private cards shown only to current player;
- sit-out settings saved;
- admin panel available only to admin;
- paused state blocks actions.
