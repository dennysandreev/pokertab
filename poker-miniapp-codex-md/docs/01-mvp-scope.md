# 01. MVP Scope

## MVP feature set

MVP включает весь scope ниже. Нельзя выкидывать leaderboard, профиль, Telegram bootstrap,
бота, историю ребаев, расчет переводов или оба режима рейтинга ради "упрощения".

Все пользовательские тексты MVP должны быть на русском. Английские строки в прототипах
считаются временными placeholder-значениями и не переносятся в продуктовый UI.

### 1. Telegram bootstrap

Пользователь открывает Mini App из Telegram.

Нужно получить:
- Telegram user id;
- username;
- first name;
- last name;
- avatar, если доступен;
- start parameter, если пользователь пришел по invite-ссылке.

### 2. Home screen

Пользователь видит:
- активные игры;
- последние закрытые игры;
- кнопку создания комнаты.

### 3. Room creation

Админ задает:
- название комнаты;
- валюту;
- сумму фиксированного ребая;
- стартовый стек;
- режим добавления ребаев.

MVP default:

```text
Игроки могут добавлять ребаи себе.
Админ может исправлять любой ребай.
```

### 4. Invite flow

После создания комнаты админ получает:
- invite link;
- кнопку "Скопировать ссылку";
- кнопку "Поделиться в Telegram".

### 5. Waiting room

До старта игры:
- игроки могут присоединяться;
- админ видит список игроков;
- админ стартует игру.

### 6. Active room — player

Игрок видит:
- название комнаты;
- сумму ребая;
- свой total buy-in;
- свои ребаи;
- главный CTA: `+ Ребай — 1 000 ₽`;
- список игроков и их общие закупы.

### 7. Active room — admin

Админ видит:
- всех игроков;
- количество ребаев;
- суммы закупов;
- кнопки `+` и `-` для корректировки;
- историю действий;
- переход к закрытию игры.

### 8. Rebuy confirmation

Каждый ребай подтверждается:
- игроком для себя;
- админом для любого игрока.

### 9. Rebuy history

Показывать:
- кто сделал ребай;
- для кого;
- сумма;
- время;
- кто отменил;
- причина отмены, опционально.

### 10. Settlement

Админ вводит финальные суммы игроков.

Приложение считает:
- total buy-ins;
- total final amount;
- difference;
- net result per player.

Итог можно закрыть только если баланс сходится.

### 11. Final results

Показывать:
- ranking;
- buy-ins;
- final amount;
- net result;
- transfer list.

### 12. Leaderboard basic

Показывать:
- total profit;
- games played;
- ROI;
- win rate;
- average result;
- poker score.

Вкладки:
- Все игроки;
- Играли со мной.

## MVP exclusions

Не реализовывать:
- платежи;
- внутренний кошелек;
- прием ставок;
- casino UI;
- депозиты;
- вывод средств;
- tournament blind timer;
- клубы;
- подписку;
- OCR;
- push-уведомления вне Telegram;
- сложную мультивалютную конвертацию.
